# Netflix_PostgreSQL_Project

# 🎬 Netflix Content Analytics | SQL Business Intelligence Project

> An end-to-end SQL analytics project that explores Netflix's global content library to uncover business insights related to content distribution, audience ratings, regional production trends, and catalog performance using PostgreSQL.

---

# 📖 Project Overview

Streaming platforms generate massive volumes of content data that can be transformed into actionable business insights. This project simulates the work of a Data Analyst by exploring Netflix's movie and TV show catalog using SQL to answer real-world business questions.

Using PostgreSQL, I performed exploratory data analysis (EDA), data cleaning, and business reporting to identify trends in content production, audience classifications, regional availability, and catalog composition.

The project demonstrates practical SQL skills commonly required in Data Analyst, Business Analyst, and Business Intelligence roles.

---

# 🎯 Business Objective

The objective of this analysis is to help stakeholders better understand Netflix's content portfolio by answering questions such as:

* What percentage of Netflix content consists of Movies versus TV Shows?
* Which content ratings dominate the platform?
* Which countries contribute the most content?
* How has Netflix's content library evolved over time?
* Which genres are most common?
* Which actors and directors appear most frequently?
* How much content has been produced in India?
* Which titles contain potentially sensitive content?

These insights can support content strategy, regional investment, licensing decisions, and audience segmentation.
